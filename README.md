# NITHdmis
A collection of tools and framework for development and fast implementation of Accessible Digital Musical Instruments dedicated to people with physical disabilities.
Written in C#, compatible with .NET Framework 4.8. To be possibly used under Windows, using Visual Studio IDE.

A full refined version of this library has still to be released.
Until further details are available, please refer to [Nicola Davanzo's Ph. D. thesis](https://air.unimi.it/handle/2434/920339) at Chapter 7 (Development > NeeqDMIs: a C-sharp HeaDMIs development framework) for an overview of the library and the proposed development paradigm.
